# Blockchain Transaction Analytics Dashboard

An interactive data visualization project built using **Tableau** to analyze ERC20 blockchain transactions and uncover key trends, wallet behavior, and transaction patterns.

**Live Website:**  
https://navyasingh24.github.io/blockchain-transaction-analytics-dashboard/


---

## Project Overview

This project focuses on analyzing blockchain transaction data (ERC20 tokens) to extract meaningful insights such as:

- Transaction trends over time  
- Most active tokens  
- Wallet activity patterns  
- Relationship between transaction value and gas fees  

The dashboard provides an **interactive interface** for exploring blockchain data and identifying patterns like high-volume wallets ("whales") and unusual transaction spikes.

---

## Objectives

- Visualize blockchain transaction data in an intuitive way  
- Identify key trends and anomalies  
- Analyze token distribution and wallet activity  
- Build a professional, interactive analytics dashboard  

---

## Dashboard Features

- **Transaction Activity Over Time** (Line Chart)  
- **Top Tokens by Transaction Count** (Bar Chart)  
- **Transaction Distribution by Token** (Treemap)  
-  **Top Active Wallets** (Bar Chart)  
- **Transaction Value vs Gas Fee** (Scatter Plot, Log Scale)  
- Interactive filters and cross-chart interactions  

---

##  Key Insights

- Certain tokens (e.g., ANON) dominate transaction activity  
- Transaction volume fluctuates significantly over time  
- A small number of wallets contribute to a large portion of transactions  
- Weak correlation observed between transaction value and gas fees
